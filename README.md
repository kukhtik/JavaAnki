# JavaAnki

Это десктопное приложение на Java (Swing) для интервального повторения (Spaced Repetition System), аналог Anki. Программа помогает запоминать ответы на вопросы, используя алгоритм уровней и "умную" проверку текста.

## Возможности

*   **Интервальное повторение:** Карточки имеют уровни (0-10). Чем лучше вы знаете ответ, тем реже выпадает карточка.
*   **Умная проверка (Fuzzy Match):** Программа сравнивает ваш ответ с правильным, игнорируя регистр и мелкие опечатки. Выводит процент сходства.
*   **Категории:** Вопросы хранятся в отдельных текстовых файлах по темам.
*   **Импорт:** Удобная система массового добавления вопросов через файл `import.txt` с автоматическим распределением по файлам категорий и удалением дубликатов.
*   **Статистика:** Подробные таблицы прогресса (общая и по темам).
*   **История:** Логирование всех ответов с возможностью поиска и фильтрации.
*   **Управление:** Полная поддержка клавиатуры (Hotkeys).

## Установка и запуск

### Требования
*   Java Development Kit (JDK) 8 или выше.

### Как запустить
1.  Убедитесь, что все файлы `.java` находятся в одной папке.
2.  Откройте терминал (консоль) в этой папке.
3.  Скомпилируйте проект:
    ```bash
    javac *.java
    ```
4.  Запустите программу:
    ```bash
    java JavaAnkiFinal
    ```

## Горячие клавиши

*   **ENTER** — Проверить введенный ответ.
*   **→ (Стрелка ВПРАВО)** — Отметить как "ВЕРНО" (после проверки).
*   **← (Стрелка ВЛЕВО)** — Отметить как "ОШИБКА" (после проверки).
*   **M** — Переключить режим (Умный алгоритм / Случайный порядок).

## Как добавлять вопросы (Импорт)

Чтобы добавить новые вопросы, не нужно создавать файлы вручную.

1.  Создайте в папке с программой файл **`import.txt`**.
2.  Запишите вопросы в следующем формате (разделитель `===` обязателен):

    ```text
    CATEGORY: Java Core
    QUESTION:
    Какие примитивы есть в Java?
    ANSWER:
    byte, short, int, long, float, double, char, boolean
    ===
    CATEGORY: SQL
    QUESTION:
    Разница между WHERE и HAVING?
    ANSWER:
    WHERE фильтрует строки до группировки, HAVING - после.
    ===
    ```

3.  Запустите программу.
4.  Перейдите на вкладку **"Статистика"**.
5.  Нажмите кнопку **"Импорт из import.txt"**.

**Что произойдет:**
*   Программа создаст файлы `decks/Java Core.txt` и `decks/SQL.txt` (если их нет).
*   Успешно добавленные вопросы **исчезнут** из `import.txt`.
*   **Дубликаты** (если вопрос уже есть в базе) будут пропущены и **удалены** из `import.txt`.
*   Если где-то ошибка в разметке, этот блок **останется** в `import.txt`, чтобы вы могли его исправить.

## Структура данных

Программа создает следующие файлы и папки:

*   **`decks/`** — Папка, где хранятся карточки (обычные `.txt` файлы). Вы можете редактировать их вручную, если нужно.
*   **`anki_stats.txt`** — Файл с прогрессом (уровни изученности карточек).
*   **`history_log.txt`** — Журнал всех ваших ответов.
*   **`import.txt`** — Файл для добавления новых карт (создается пользователем).

---

## Структура кода

Проект разбит на модули для удобства поддержки:

1.  **`JavaAnkiFinal.java`** — Точка входа (Main).
2.  **`MainFrame.java`** — Графический интерфейс (GUI), вкладки, таблицы.
3.  **`Card.java`** — Модель карточки (хранит вопрос, ответ, уровень).
4.  **`DeckManager.java`** — Логика работы с колодой (загрузка, сохранение прогресса, выбор следующей карты).
5.  **`DeckDistributor.java`** — Логика импорта из `import.txt` и распределение по файлам.
6.  **`SmartChecker.java`** — Алгоритм сравнения строк (расстояние Левенштейна).
7.  **`HistoryManager.java`** — Запись и чтение истории ответов.
8.  **`KeyController.java`** — Обработка нажатий клавиш.
9.  **`AppLogger.java`** — Красивый вывод логов в консоль.
