CATEGORY: Stream API
QUESTION:
Промежуточные vs Терминальные операции.
ANSWER:
Промежуточные (filter, map, sorted): ленивые, возвращают Stream, не выполняются без терминальной.
Терминальные (collect, forEach, count): запускают выполнение стрима, возвращают результат или void.
